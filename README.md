# 🏠 Housing Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts California housing prices using Machine Learning. The project includes data preprocessing, exploratory data analysis (EDA), visualization, model training, evaluation, and model saving using Python and Scikit-Learn.

---

## 🚀 Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Correlation Heatmap
- Pairplot Visualization
- Linear Regression Model
- Model Evaluation
- Model Saving using Pickle

---

## 📊 Dataset

Dataset: California Housing Dataset

The dataset contains housing-related features such as:

- Median Income
- House Age
- Average Rooms
- Average Bedrooms
- Population
- Average Occupancy
- Latitude
- Longitude

Target:
- Median House Value

## ▶️ How to Run

Clone the repository

```bash

```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the notebook

```bash
jupyter notebook
```

or run the Python script

```bash
python housing_price_prediction.py
```
